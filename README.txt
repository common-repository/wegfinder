With wegfinder buttons, your website visitors will find all the routes (public transport, car, bicycle, sharing, ...) to the places and events in Austria indicated on your website.Enables the previous "classic" editor and the old-style Edit Post screen with TinyMCE, Meta Boxes, etc. Supports all plugins that extend this screen.

== Description ==

(English version below)

Wie wohin? Zeige deinen Besuchern alle Wege zu den von dir ausgewählten Orten!
Ganz gleich ob mit den Öffis, Auto, Fahrrad, Car Sharing, etc. - wegfinder findet alle Wege.

So gehts:
1. Erstelle ein Ziel. Dabei kann es sich um eine Stadt, eine Adresse, eine Station oder einen POI in Österreich handeln
2. Wähle ein Button Design, oder gestalte deinen eigenen Button.
3. Wähle einen Button im Block-Editor oder im Classic-Editor aus um ihn auf einer Seite oder in einem Beitrag einzubinden.

Klickt ein Besucher auf einen Button, findet wegfinder alle Wege zum Ziel. Das Ergebnis wird entweder auf wegfinder.at angezeigt, oder direkt in der wegfinder App (Android, iOS), wenn diese installiert ist.

Tipps: 
* Du kannst optional auch einen Ankunftszeitpunkt für deine Ziele festlegen. Das ist besonders praktisch für Veranstaltungen.
* Du kannst die Größe und Farbe der Buttons individuell anpassen.
* Du kannst Ziele einfach ändern, alle Buttons werden dann automatisch aktualisiert.


Es handelt sich um das offizielle wegfinder Plugin das von iMobility GmbH entwickelt wird.


(English version)

How to get there? Show your visitors all the ways to the places you have chosen!
No matter if with the public transport, car, bicycle, car sharing, etc. - wegfinder finds all paths.

That's it:
1. Create a destination. It can be a city, an address, a station or a POI in Austria.
2. Choose a button design, or design your own button.
3. Simply select a button in the block editor or in the classic editor to embed it on a page or in a post.

When a visitor clicks on a button, wegfinder finds all paths to the destination. The result is displayed either on wegfinder.at or directly in the wegfinder app (Android, iOS), if it is installed.

Tips: 
* You can also optionally specify an arrival time for your destinations. This is especially useful for events.
* You can customize the size and color of the buttons.
* You can easily change goals, all buttons will be updated automatically.

This is the official wegfinder plugin developed by iMobility GmbH.
